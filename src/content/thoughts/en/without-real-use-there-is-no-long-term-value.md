> Investment is not speculation in the air; what ultimately supports price is use value underneath.

Lecture 051 argues that without use demand, there is no investment demand. The discussion may begin with housing, but the principle applies to any asset: long-term value cannot be separated from real use.

People can speculate on price movement in the short run. They may buy because they believe someone else will pay more later. But if no one ultimately wants to use the asset, live in it, rent it, consume it, or benefit from it, the price is floating without foundation.

Housing has investment value because it has use value. People need places to live, work, study, and access opportunities. If a house is in a place with real demand, its investment value has support. If it is empty, unusable, or detached from real need, price appreciation becomes fragile.

The same is true for companies, products, platforms, and collectibles. A business without users, a product without usage, and a platform without real activity may attract attention for a while, but long-term value requires someone to keep deriving benefit from it.

This does not mean every valuable thing must be practical in a narrow sense. Art, memory, status, and symbolism can all be real uses if people genuinely value them. The key is that some real human demand must remain.

For personal investment, the question is not only whether the price has risen before. It is whether there is durable use underneath. Who needs this? Why will they continue paying? What value does it create if speculation cools down?

Without real use, there is no long-term value. Price can run ahead for a while, but eventually it must return to what people actually need, want, and use.
